a CKEditor plugin to easily add code into your articles

What pbckcode does

This is a CKEditor plugin that provides an IDE-like features and creates the good syntax to load your syntax highlighter. The IDE-like features are possible owing to the ACE Editor plugin.

What pbckcode does not

This plugin will not (and never) highlight the code in the ckeditor textarea. It only provides an easy way to write code for your posts and prepare the needed markup for your view.

Story behind the plugin

It came from my personal need. I am a (little) blogger and I wanted to write code articles and tutorials. However I did not found any decent plugins to add code blocks into posts so I decided to create mine.

pbckcode is intended for writer, bloggers, coders, developers, everybody ! so feel free to try it !
